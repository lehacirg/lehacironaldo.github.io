About

I am a programmer, passionate about Unity, 3D Modelling and Graphic Design with a keen interest in video game development.

Growing up in Timisoara, Romania, I was always passionate about programming, starting my first projects in Macromedia Flash Player, making dress-up games for stickmen in ActionScript, it lit my passion for creating, be it anything from YouTube channels, art, music to programming in Unity 3D / C#, as of today.

Studies

XAMK Finland / Cambridge Academy for Digital Games and Innovation

Before starting college, I thought I would dive in myself in the world of programming, using this course provided by XAMK Finland / CADGI on "Introduction to Video Games Creation". It took me a whopping ~700 hours to complete, being a complete beginner in the world of programming as a whole, not mentioning the rest of the tools we used during the course.

During the course, (accessible here), I learned how to use different tools such as C#, Unity 3D, Visual Studio, Blender, Kryta and Google in order to introduce myself into the complicated world which is game development.

I have also learned about project management, market research, AI functionality, good work and coding ethics, debugging, etc. (For detailed information, check out my diploma).

University of West Timisoara, Faculty of Economics and Business Administration, Informatical Economics Major

Majoring in Informatical Economics as a programmer, I thought it would help me better understand certain concepts about companies and entrepreneurship as a whole, while still keeping up with technical elements, such as .NET C#, Microsoft Access and SQL databases.

In the first year, I took a class called "Introduction to Programming in C#", in which I consolidated my former studies at XAMK/CADGI, by using C# in a different setup, .NET Framework. I've learned to build apps using the wide variety of tools .NET has to offer.

In the second year, I am currently working on learning Microsoft Access and SQL databases.

My Projects

Programming

XAMK / CADGI

Game Development Course

Text-Based Adventure Game, "The Return"

In this module about making a text-based adventure game, I learned more advanced project setup, world data structures, game controller, user input, systems of actions, as well as items, item-to-player interactions and NPC conversations. Overall, it was a fun game to create, though I couldn't help myself but to try and improve its graphics, be it fancy fonts or just better colors, given the base visual simplicity of this first project.

Interactive Card-Combat Game, "Merlin's Demon War"

During this module, I learned how to import graphics and GUI, create card prefabs, code the prefabs so they instantiate properly, create a player and player controller, coroutines so the cards can deal, basic animations, basic sprite FX, bugfix a way bigger project than I was initially used to, add SFX. This second project had more advanced graphics, which felt refreshing.

Mobile Android/iOS Game Implementation, "Snake Game"

Throughout this module, I learned how to create a Unity build for Android/iOS, how to test using my personal smartphone in real-time, endless game progression, collisions, more GUI, sounds and smartphone app deployment with icons and splash screens. In the course of this third and final project, I was already ecstatic about going to create my own games.

Final year one exam application, made for "Introduction to C# Programming" course

During this course, we were given free will in the terms of designing an entire application, using the tools we learned about inside .NET Framework.

I came up with an application that is user:password protected at first, has a rudimentary calculator, and a gross-to-net salary calculator, based on types of, and tax percentage applied in a selection of different years in Romania, as well as three mini-games (I couldn't help myself not to) : Dice Roll, Heads or Tails and Long/Middle/Short straw game.

Personal Projects

(WIP) 2.5D Hardcore Platformer Game, "Hovering Moonmen"

Hovering Moonmen is my first dare at single-handedly creating an entire game using the Unity 3D engine. I used Blender 3D and Kryta to create the assets used in the game, such as materials, models, props, GUI, UI, HUD, etc. The core mechanics are simple, it's basically "Mario from Hell", with a higher level of graphics. It uses walking, running, jumping, double jumping and a flashlight mechanic used to light your way around the levels. It was a great project which helped me assimilate information about more advanced game development,

(WIP) 3D Multiplayer Psychological Horror Game, "Descent : Narrow Escape"

While I can't say much about the development of this game, since I are currently still working on the final concept, it's a multiplayer FPS psychological horror game, which is going to feature eerie gameplay mechanics such as an AI element which actively hunts players, while they have to gather certain objects in order to fulfill tasks, to help them escape, and other features such as faux water simulation, advanced lighting systems, professional art direction and high-fidelity graphics.
